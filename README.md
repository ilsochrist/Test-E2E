# Test-E2E


This is a simple project that i did to create and learn about e2e tests on Web APIs with Jest and Node.js.

<h3>## Used tools:</h3>
<ul>
<li>JavaScript</li>
<li>Jest</li>
<li>Node.js</li>
</ul>

## Available Scripts

In the project directory, first you have to install node modules in terminal:

### `npm i`

Now you can run:

### `npm test`
